# Real-time Ice Cream Translator

Система потокового перевода аудио- и видеоконтента в режиме реального времени.

Проект предназначен для захвата системного аудиопотока, его обработки с помощью готовой LLM и генерации переведенной речи параллельно оригинальному звуковому ряду с минимальной задержкой (latency).

## Механизм работы

Приложение функционирует как промежуточный слой между источником звука (браузер, плеер, системный микшер) и устройством вывода. Основная логика построена на концепции потокового инференса: данные обрабатываются короткими сегментами (чанками) без необходимости ожидания завершения всей фразы или сохранения временных файлов на диск.

В качестве базовой модели используется [SeamlessM4T-v2-large](https://huggingface.co/facebook/seamless-m4t-v2-large) от Meta, выполняющая прямую задачу Speech-to-Speech Translation (S2ST).

## Базовый функционал (Baseline)

* **System Audio Capture (Loopback):** Прямой захват входящего аудиопотока из операционной системы без использования внешнего микрофона.
* **Streaming Processing:** Потоковая передача данных в модель, исключающая задержки, связанные с операциями ввода-вывода (I/O).
* **Multilingual Support:** Поддержка широкого набора языковых пар, включая китайский, английский и русский языки.
* **Low Latency Inference:** Оптимизация размеров сегментов аудио для достижения баланса между качеством перевода и скоростью вывода.

## Схема работы пайплайна

Обработка данных в системе разделена на следующие этапы:

1. **Audio Capture:** Перехват аудиопотока через системные API в моно-формате.
2. **VAD (Voice Activity Detection):** Детекция голосовой активности для фильтрации тишины и фоновых шумов, что снижает вычислительную нагрузку на модель.
3. **Buffering & Preprocessing:** Нарезка потока на окна фиксированной длины и ресемплирование до 16 кГц (согласно требованиям входного тензора SeamlessM4T).
4. **Model Inference:** * **ASR (Automatic Speech Recognition):** Распознавание исходной речи.
   * **NMT (Neural Machine Translation):** Перевод текста на целевой язык.
   * **TTS (Text-to-Speech):** Синтез аудио на целевом языке.
5. **Audio Alignment:** Динамическая синхронизация синтезированного аудио с оригинальным потоком.
6. **Playback:** Вывод итогового результата на стандартное устройство воспроизведения.
